Generics

Generics

Allows us to have a variable data types

 struct Point<T> {
     x: T,
     y: T
 }

Usage let p1: Point<i32> = Point {x:6, y: 8}; let p2: Point<f64> = Point {x:3.25, y:8.43}

In Enums

enum Colors<T> {
    Red(T),
    Blue(T),
    Green(T)
}

usage:

let color1 = Colors_with_gen::Blue("blue");

let color2 = Colors_with_gen::Red(255);

Multi Generics

 struct Point<T, V> {
     x: T,
     y: V
 }

usage: let p2: Point<i32, f32> = Point { x: 6, y: 8.44 };
